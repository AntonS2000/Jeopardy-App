# Тестирование новой функциональности сигналов

## Описание изменений

1. **Красная круглая кнопка** добавлена в панель игрока:
   - Расположена по центру страницы
   - Анимация нажатия (увеличение/уменьшение при наведении и клике)
   - Блокируется после первого нажатия
   - Сигнализирует, что игрок готов отвечать на вопрос

2. **Жёлтые индикаторы** добавлены в карточки:
   - У игроков: показывают статус сигналов других игроков
   - У администратора: показывают статус сигналов всех игроков
   - При нажатии на кнопку игроком индикаторы становятся золотыми (#ffd700)
   - У администратора есть возможность разблокировать сигнал, нажав на желтый индикатор

3. **Серверная логика**:
   - Отслеживает активный сигнал
   - Предотвращает одновременное нажатие несколькими игроками
   - Обрабатывает разблокировку администратором

## Тест-кейсы

### Тест 1: Нажатие красной кнопки игроком
1. Подключить игрока к сессии
2. Нажать красную кнопку
3. Проверить:
   - Кнопка заблокировалась
   - Жёлтый индикатор стал золотым у всех участников
   - Другие игроки не могут нажать свои кнопки

### Тест 2: Разблокировка сигнала администратором
1. После активации сигнала игроком
2. Администратор нажимает на желтый индикатор
3. Проверить:
   - Жёлтый индикатор стал обычным цветом
   - Красные кнопки у игроков снова активны

### Тест 3: Повторное нажатие после разблокировки
1. После разблокировки администратором
2. Любой игрок может снова нажать красную кнопку
3. Проверить:
   - Система снова работает как в Тесте 1

## Технические детали

### HTML-изменения
- Добавлены новые CSS-классы для стилей кнопки и индикаторов
- Добавлены элементы для красной кнопки и желтых индикаторов
- Добавлены обработчики событий

### JavaScript-изменения
- Новые функции для обработки сигналов
- Обновлены функции обновления индикаторов
- Добавлена логика блокировки/разблокировки кнопок

### Серверные изменения
- Добавлена глобальная переменная для отслеживания активного сигнала
- Новые обработчики сокет-событий
- Обновлены существующие обработчики для передачи информации о желтых индикаторах
